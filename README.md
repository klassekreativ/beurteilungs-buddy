# Beurteilungsbeitrag-Assistent (OVP NRW)

Ein webbasiertes Tool zur strukturierten Erstellung von Beurteilungsbeiträgen gemäß **§ 16 (2) OVP NRW**. Dieses Tool hilft Ausbildungslehrkräften und Schulleitungen dabei, Beobachtungen systematisch zu erfassen und daraus professionelle Textbausteine oder Prompts für KI-Unterstützung zu generieren.

##Funktionen

- **Strukturierte Erfassung:** Abfrage aller 5 Handlungsfelder (U, E, L, B, S) basierend auf den Kernkompetenzen der OVP NRW.
- **Flexibler Output:** Erzeugung von fertigen Fließtext-Bausteinen oder optimierten Prompts für KI-Modelle (z. B. Claude oder ChatGPT).
- **Anpassung an den Ausbildungsstand:** Berücksichtigung des 1. oder 2. Ausbildungsabschnitts in der Formulierung.
- **Anonymisierungs-Option:** Export von KI-Prompts ohne Klarnamen für maximalen Datenschutz.
- **iPad-optimiert:** Voll funktionsfähig als Web-App auf mobilen Endgeräten.

## Datenschutz 

Dieses Tool wurde mit Fokus auf Datensparsamkeit entwickelt:
- **Kein Server:** Es gibt kein Backend. Alle Eingaben werden ausschließlich lokal im Browser verarbeitet.
- **Keine Speicherung:** Sobald der Browser-Tab geschlossen wird, werden (in der Standard-Version) keine Daten dauerhaft gespeichert.
- **KI-Sicherheit:** Bei der Nutzung von KI-Prompts wird empfohlen, die anonymisierte Export-Funktion zu nutzen.

## Nutzung

Du kannst das Tool direkt über GitHub Pages nutzen:
**https://klassekreativ.github.io/beurteilungs-buddy/**

Alternativ kann die `index.html` heruntergeladen und lokal auf jedem Rechner mit einem modernen Browser geöffnet werden.
